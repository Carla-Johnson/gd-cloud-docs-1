Install Jenkins

Install Jenkins - Arch Linux

Difficulty: 1
Time: 10 minutes

Jenkins is an extensible open source continuous integration server used for building software projects and monitoring job execution.

Install and start Jenkins

  1. Install Jenkins.
    sudo pacman -Syu jenkins
  2. Start Jenkins.
    sudo systemctl start jenkins
  3. Open a browser and navigate to your Jenkins server.
    http://Your IP Address:8090/

Welcome to Jenkins!

Configuring Jabber XMPP server in Ubuntu and Debian, XMPP

Install and configure a Jabber XMPP server - Ubuntu, Debian

Difficulty: 2

Time: 20 Minutes

Ejabberd is written in Erlang and known as a high performing Jabber/XMPP server. It has its own web-based management interface. This article explains the installation of Jabber XAMPP server in Ubuntu and Debian. You’ll need access to root privileges to complete these steps.

Installation

  1. Use following command to install the server along with its component.
    sudo apt-get install ejabberd
  2. We have installed the server but we need to change a few configuration settings.

Configuration Settings

  1. We need to edit the configuration file. It resides at /etc/ejabberd/ejabberd.cfg. Use any editor and open this file.
  2. We need to add the host name we want server to use , so swap
    {hosts, ["localhost"]}

    with

    {hosts, ["localhost","myserver.com"]}
  3. Now we need to create an admin user so swap
    {acl, admin, {user, "", "localhost"}}

    with

    {acl, admin, {user, "my_admin", "myserver.com"}}
  4. Now restart the service with following command
    sudo service ejabberd restart
  5. To set password for admin user use following command
    sudo ejabberdctl register my_admin myserver.com password

Next Steps

In this article, we reviewed the steps involved with the installation and configuration of jabberd server. To change any other configurations you’ll need to log in to the web-based management panel. In your browser you can go to http:// myserver.com:5280/admin and enter admin credentials.

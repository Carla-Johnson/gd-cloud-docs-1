Configure Themes and Settings for Ghost - Arch Linux

Difficulty: 2

Time: 30 minutes

Ghost is an open-source project for blogging and blogger. It provides a simple blogging platform, fast to setup and easy to setup. Besides, it provides good customization options to the users. This article will assist you with the steps to set up a Ghost blogging platform using NGINX on your Ubuntu platform. The themes used in this tutorial are available on internet as freeware from GhostMarketplace.

Change the Ghost Themes

  1. Login to your instance and change directory to ghost theme using the below command
    cd /var/www/ghost/content/themes/
  2. Install git and gitcore if not already installed
    sudo pacman -S git
  3. Clone the theme from the GitHub project
    sudo git clone https://github.com/polygonix/N-Coded.git
  4. Change the ownership of the ghost user
    sudo chown -R http:http N-Coded

    In case you get an invalid user error, then please create ghost user then retry.

  5. Start npm services
    sudo npm start --production

Change the Settings

  1. Open your browser and type http://your_IP_address/ghost/settings/general in the address bar
  2. It will redirect you to the time setting page. Fill the required information about your blog and credentials here. You will see the first setup form as:
  3. In the general setting page, scroll down to see theme label. Select the theme from the drop down and click Save
  4. Refresh your ghost public home page to see the blog with updated theme

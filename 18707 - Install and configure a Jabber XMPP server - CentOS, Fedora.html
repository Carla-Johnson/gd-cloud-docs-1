Setting up Jabber XMPP Server on CentOS and Fedora, XMPP

Install and configure a Jabber XMPP server - CentOS, Fedora

Difficulty: 2

Time: 30 Minutes

This article explains installation and configuration of ejabber on CentOS and fedora

Jabber is simple, mature and powerful massaging protocol which developed for real-time massaging between two more people based on text typed. Jabber is used by enterprice for internal communication, developer to improve enlarge programming skill, hosting provider for high available and stable communication services, individual for chatting fun.

Note: You will need access to sudo privileges to complete all steps.

Step 1 - Download ejabber installer

  1. wget http://www.process-one.net/downloads/downloads-action.php?file=/ejabberd/14.12/ejabberd-14.12-linux-x86_64-installer.run -o ejabberd-14.12-linux-x86_64-installer.run

  2. Change file name

    mv downloads-action.php\?file\=%2Fejabberd%2F14.12%2Fejabberd-14.12-linux-x86_64-installer.run ejabberd-14.12-linux-x86_64-installer.run

    Change permission

    sudo chmod +x ejabberd-14.12-linux-x86_64-installer.run

  3. Execute Installer

    d. sudo ./ ejabberd-14.12-linux-x86_64-installer.run

Step 2 - Select Language

  1. After start installer, it will ask for Language.
    select language English on Number [2]

Step 3 - Next, ejabber installer show you eula agreement

  1. User agreement press [enter] till end

Step 4 - Accept agreement

  1. Do you accept this license? [y/n]: Y [Press Y]

Step 5 - Select installation directory

  1. Installation Directory [/opt/ejabberd-14.12]: [press enter]

    if you want to keep default directory press enter, you can specify another path as per your convenience.

Step 6 - Select installation directory

  1. Installation Directory [/opt/ejabberd-14.12]: [press enter]

Step 7 - Select domain here

  1. ejabberd server domain [bpcs-centos]

    Type FQDN or public IP or localhost or any internal IP from which we can access it.

Step 8 - Setup administrator credentials

  1. Administrator username [admin]:

    In above prompt we can choose any username, it will your administrator

    Tips: Choose different username tcd /opt/ejabberd-14.12/bin/o make it some secure

  2. After choosing username, next prompt will for password. Choose password very carefully and go ahead.

Step 9 - Setup administrator credentials

  1. After submit password, installer will prompt for cluster , Press 'N' as of now we don't require.

    promoted massage example Cluster [y/N]:N

Step 10 - Final step for installation

  1. After key Y installer will start the installation process it will take some time to finish it.

    Do you want to continue? [press Y]

  2. Next it will prompt for view readme

    View Readme file? [Y/n]: n

    Press n because don't think we need to read it. Of course you can press Y if you want to read.

Step 11 - Starting Ejabber services

  1. Now our ejabber protocol is ready installed, next will start it to use it following is command to start ejabber protocol.

    cd /opt/ejabberd-14.12/bin/ sudo ./ejabberdctl start

Step 12 - Verify ejabber service

  1. Now ejabber massage service is started lets verify it by using following command.

    sudo netstat -plunt | grep 5280

  2. If you found our as below, congratulation you have successfully installed jabber instant messaging service

    tcp 0 0 0.0.0.0:5280 0.0.0.0:* LISTEN 27744/be

Step 14 - Browser

  1. Now open browser, type http://your_ip:5280/ after hit enter. It will prompt for authentication credentials, Enter those credential will we have entered during installation time. After successfully login it will show screen like below

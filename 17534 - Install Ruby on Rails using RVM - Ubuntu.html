Steps to install Ruby on Rails using Ruby Version Manager (RVM) in Ubuntu, Ruby, Rails

Install Ruby on Rails using RVM - Ubuntu

Difficulty: 1

Time: 30 minutes

Ruby on Rails® is a well-known open source stack for web application development. With Ruby Version Manager (RVM) you can easily manage multiple Ruby application environments and efficiently switch between them. This article will assist you on how to install Ruby on Rails in Ubuntu® 14.04 VPS using RVM.

Prerequisites

To get started, you'll need the following prerequisites:

  • RVM (Ruby Version Manager)
  • SQLite
  • MySQL™ (Optional)
  • Apache2® (Optional)

Install RVM and Ruby

  1. Install RVM by running the following commands:
    sudo apt-get install libgdbm-dev libncurses5-dev automake libtool bison libffi-dev
    sudo curl -L https://get.rvm.io | bash -s stable
  2. After you run the above command, there are more dependencies that needs to be fixed, so run the following commands:
    sudo curl -sSL https://rvm.io/mpapis.asc | gpg --import -
    sudo curl -L https://get.rvm.io | bash -s stable
    source ~/.rvm/scripts/rvm
    rvm install ruby
    ruby -v
  3. Ruby installs the contents for all the packages. To inhibit this, type the following command:
    sudo echo "gem: --no-ri --no-rdoc" > ~/.gemrc
  4. Install the bundler.
    gem install bundler
  5. To prevent any dependencies of Rails libraries, install NodeJS by typing:
    sudo add-apt-repository ppa:chris-lea/node.js
    sudo apt-get update
    sudo apt-get install nodejs

Install Rails

  1. Use gem to install Rails. This will take some time to install.
    gem install rails

    After completion, your environment will be created.

  2. Check your Rails version, type:
    rails -v
  3. Create the project directory for your Rail project, by typing the following commands:
    sudo mkdir -p /usr/local/src/rail_project
  4. Move to the directory just created, and grant all the required access using the following commands:
    cd /usr/local/src/rail_project
    sudo chmod 777 /usr/local/src/rail_project
    rails new HelloWorld
    cd HelloWorld
    rails server

    Now your Rails server will start running.

  5. Bind your Rails application using the following command:
    rails server --binding=[your public IP]

Your new application is ready, and can be accessed by visiting your_public_IP_or_domain:3000 (3000 is default port).

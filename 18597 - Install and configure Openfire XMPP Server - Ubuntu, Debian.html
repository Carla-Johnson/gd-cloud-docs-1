Install and configure Openfire XMPP Server - Ubuntu, Debian

Difficulty: 2

Time: 20 minutes

In this article, we will assist you to understand the installation and configuration of Openfire XMPP Server on Ubuntu and Debian.

Understanding XMPP

Extensible Messaging and Presence Protocol (XMPP) is a XML-based communication protocol for middleware. It enables the near-real-time exchange of structured yet extensible data between any two or more network entities.

Understanding Openfire

Openfire enables collaboration in real time and provides a tightly secured and high performing server. You can use Open Source Apache License for Openfire. You can easily set it up and configure using Java.

Note: Openfire doesn't allow any chat options.

Step 1: Installing Openfire

  1. Update the operating system
    sudo apt-get update
  2. If java is installed in your system then check the version of java using the following command
    Java -version
  3. If java is not installed in your system then run the following command.
    sudo apt-get install default-jre
  4. Download the Openfire
    sudo wget -O openfire.deb http://www.igniterealtime.org/downloadServlet?filename=openfire/openfire_3.8.2_all.deb
  5. This will initiate the download of the Openfire Debian package to openfire.deb. Next, install the package:
    sudo dpkg --install openfire.deb

    Openfire has been successfully installed in your system.

  6. Now visit http:// your_IP/9090 on any browser, then Openfire wizard

  7. The first page will allow you to choose your language while the second page is server setting page where you can define your domain name or IP. See the below image:
  8. Once you are done with the setup, login as an Admin on Openfire login page
  9. To see server information, click on the 'server manager' tab


  10. If you want a secure connection to the server then go to Server Settings -> Registration first and then login and disable the required options.

Step 2: Execute and Stop Openfire

  1. If you need to stop Openfire, then type the following command:
    sudo /etc/init.d/openfire stop
  2. To start Openfire, type:
    sudo /etc/init.d/openfire start

Next Step

By using all the above steps you'll be able to install and configure Openfire XMPP server on Ubuntu and Debian.

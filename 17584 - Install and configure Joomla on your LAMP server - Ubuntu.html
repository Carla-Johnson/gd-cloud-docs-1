Installation of Joomla on your web server.

Install and configure Joomla on your LAMP server - Ubuntu

Difficulty: 1
Time: 5 minutes

Joomla is an open source content management system that helps you build websites and online applications. Joomla is built in PHP and makes use of a database system like MySQL.

This article walks you through the installation of Joomla on your Ubuntu server. Before getting started, you need to have LAMP (Linux, Apache, MySQL, PHP) set up on your server. For instructions, see installing LAMP stack.

Download Joomla

  1. Before you download the installation files, create a directory.
    sudo mkdir temp
  2. Go to that directory.
    cd temp
  3. Download the latest version of Joomla from the Joomla website.
    sudo wget http://joomlacode.org/gf/download/frsrelease/17410/76021/Joomla_2.5.7-Stable-Full_Package.tar.gz
  4. Unzip the downloaded package, moving it to the default Apache directory.
    sudo tar zxvf Joomla_2.5.7-Stable-Full_Package.tar.gz -C /var/www/html
    If the command tar is not found, install tar.
    sudo apt-get install tar

Grant permissions

  • Alter the permission on the configuration file. You can change it back after the installation is complete.
    sudo touch /var/www/html/configuration.php
    sudo chmod 777 /var/www/html/configuration.php

Create Joomla DB and User

  1. Login to MySQL.
    sudo mysql -u root -p
    When prompted, enter the MySQL root password that you set during the MySQL installation.
  2. Create a database with any name.
    CREATE DATABASE testdb;
    The following displays:
    Query OK, 1 row affected (0.00 sec)
  3. Create a user.
    CREATE USER newuser@localhost;
  4. Set the password for the newly created user.
    SET PASSWORD FOR newuser@localhost= PASSWORD("pass");
  5. Grant privileges to the newly created user.
    GRANT ALL PRIVILEGES ON testdb.* TO newuser@localhost IDENTIFIED BY 'pass';
  6. Refresh MySQL.
    FLUSH PRIVILEGES;
  7. Exit the MySQL shell.
    exit
  8. Restart Apache.
    sudo service apache2 restart

Install Joomla

  1. In your browser, navigate to http://server-ip-address/installation. The Joomla installation page displays.
  2. On the Pre-installation Check and License page, click Next.
  3. On the Configure the database connection settings page, enter the credentials you created in the previous section.
  4. Enter configuration information for your website, such as the name of the site, and the email and password for administrator.
  5. Click Install Sample Data.
    To complete the installation, you will be asked to remove the installation folder.
  6. In your console, remove the installation folder, and change the permission on the configuration file.
    sudo rm -rf /var/www/html/installation/
    sudo chmod 755 /var/www/html/configuration.php
  7. In your browser, navigate to http://server-ip-address/index.php or domain name. The new Joomla page displays.

Next steps

Now that you have set up a Joomla site, you can install Joomla Console to manage your Joomla site by command line.

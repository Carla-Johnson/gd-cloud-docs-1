Install Jenkins

Install Jenkins - Debian, Ubuntu

Difficulty: 1
Time: 10 minutes

Jenkins is a an extensible open source continuous integration server used for building software projects and monitoring job execution.

Install and start Jenkins

  1. Get the Jenkins installation key.
    sudo sh -c 'echo deb http://pkg.jenkins-ci.org/debian binary/ > /etc/apt/sources.list.d/jenkins.list'
  2. Get the Jenkins installation package.
    wget -q -O - https://jenkins-ci.org/debian/jenkins-ci.org.key | sudo apt-key add -
  3. Update your apt repository.
    sudo apt-get update
  4. Install Jenkins.
    sudo apt-get install jenkins
  5. Start Jenkins.
    sudo /etc/init.d/jenkins start
  6. Open a browser and navigate to your Jenkins server.
    http://Your IP Address:8080/

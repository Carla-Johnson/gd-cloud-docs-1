Setting up Jabber XMPP Server on CentOS and Fedora, XMPP

Install and configure Jabber XMPP server - CentOS, Fedora

Difficulty: 2

Time: 30 Minutes

This article explains installation and configuration of ejabber on CentOS and Fedora.

Jabber is a simple, mature and powerful messaging protocol which was developed for real-time messaging between two or more people,s based on the text typed. Jabber is used by enterprises for internal communication, developers for improving programming skills, hosting provider for highly available and stable communication services, individual for chatting.

Note: You will need access to sudo privileges to complete all steps.

Step 1 - Download ejabber installer

  1. Run the following command to download ejabber:

    wget http://www.process-one.net/downloads/downloads-action.php?file=/ejabberd/14.12/ejabberd-14.12-linux-x86_64-installer.run -o ejabberd-14.12-linux-x86_64-installer.run

  2. Change file name

    mv downloads-action.php\?file\=%2Fejabberd%2F14.12%2Fejabberd-14.12-linux-x86_64-installer.run ejabberd-14.12-linux-x86_64-installer.run

    Change permission

    sudo chmod +x ejabberd-14.12-linux-x86_64-installer.run

  3. Execute Installer

    d. sudo ./ ejabberd-14.12-linux-x86_64-installer.run

Step 2 - Select Language

  1. After starting the installer, it will ask for language.
    select language English on Number [2]

Step 3 - Next, ejabber installer shows you eula agreement

  1. User agreement press [enter] till end

Step 4 - Accept that agreement

  1. Do you accept this license? [y/n]: Y [Press Y]

Step 5 - Select installation directory

  1. Installation Directory [/opt/ejabberd-14.12]: [press enter]

    If you want to keep default directory press enter, you can also specify another path as per your convenience.

Step 6 - Select installation directory

  1. Installation Directory [/opt/ejabberd-14.12]: [press enter]

Step 7 - Select domain here

  1. ejabberd server domain [bpcs-centos]

    Type FQDN or public IP or localhost or any internal IP from which we can access it.

Step 8 - Setup administrator credentials

  1. Administrator username [admin]:

    In the above prompt you can choose any username, that will become your administrator.

    Tip: Choose different username to make it more secure.

  2. After choosing username, next it will prompt for password. Choose password very carefully and go ahead.

Step 9 - Setup cluster

  1. After submitting password, installer will prompt for cluster. Press 'N' as we don't require it now.

    promoted massage example Cluster [y/N]:N

Step 10 - Perform the final step for installation

  1. To continue, press Y. The installer will start the installation process and it will take some time to finish it.

    Do you want to continue? [press Y]

  2. Next it will prompt for view readme

    View Readme file? [Y/n]: n

    Press N if you don't want to read it and press Y if you want to read.

Step 11 - Start Ejabber services

  1. Now our ejabber protocol is ready to be used. Use the following command to start ejabber protocol:

    cd /opt/ejabberd-14.12/bin/ sudo ./ejabberdctl start

Step 12 - Verify ejabber service

  1. Now ejabber message service is started. Let us verify it by using the following command:

    sudo netstat -plunt | grep 5280

  2. If you find output as below, congratulations you have successfully installed jabber instant messaging service.

    tcp 0 0 0.0.0.0:5280 0.0.0.0:* LISTEN 27744/be

Step 14 - Browser

  1. Now open your browser, type http://your_ip:5280/ and hit Enter. It will prompt for authentication credentials, enter those credentials which you have entered during installation time. After successfully logging in, it will show the screen as below
